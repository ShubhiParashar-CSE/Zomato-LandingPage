# Zomato Landing Page Clone

A static landing page that mimics Zomato for discovering food and drinks in Delhi NCR.

## Tech Stack
- HTML, CSS
- Font Awesome for icons
- Google Fonts (Poppins)

## Features
- Hero section with navigation and search functionality
- Sections for ordering online, dining, and nightlife options
- Collections of trending restaurants and special offers
- Popular localities listing
- Get the app section with download options

## How to Run
Just open the `index.html` file in any web browser to view the landing page.

## What I Learned
This project helped me practice building responsive layouts with HTML and CSS, integrating external fonts and icons, and structuring a clean, user-friendly landing page.
